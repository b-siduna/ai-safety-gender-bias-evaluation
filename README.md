# ai-safety-gender-bias-evaluation
A prompt-based comparative evaluation of gender bias in ChatGPT and Gemini using an AI safety assessment framework.
# Overview
This repository contains a technical report evaluating gender bias in ChatGPT and Gemini using an AI safety assessment framework.
## Objective
The study compares responses generated by large language models to gender-related prompts, focusing on fairness, bias, and responsible AI.
